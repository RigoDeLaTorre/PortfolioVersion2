Trucking Buddy- Logistics App

Overview

This is a logistics app for the trucking industry.
You have the ability to either use my login or create a new company profile, where you will be able to add drivers, customers, loads.
The forms are built with redux-forms and have field validation as well as normalization.


Please be patient, as its hosted on Heroku. Takes about 15 seconds for the server to "wake up".
